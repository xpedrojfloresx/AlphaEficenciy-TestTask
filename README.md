# Better Weigh Inc. — Landing Page

Test task for **Alpha Eficency**, application for the **Web Developer** position.

Responsive landing page built from a design, using vanilla HTML, CSS and JavaScript.

## Features

- **Responsive design** from 1920px down to 320px.
- **Hamburger menu** working on tablet and mobile.
- **Countdown timer** ("Your offer ends in...") that resets automatically.
- **Interactive FAQ accordion**.
- **Design details**: decorative circles, countdown card with a glassmorphism effect, and sections with overlapping images.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, media queries)
- JavaScript (vanilla)
- Fonts: Libre Baskerville and Montserrat (Google Fonts)

## Structure

